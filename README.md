# Family Assistant Web Application
Simple Django application which includes calendar and distance calculator

### Oveview
The application servers as a mean for families to track their activities, plan family trips or just share some thoughts about trips from the past.

### Technologies
* Django for backend alongside SQLIte3 database.
* HTML, Bootstrap and vanilla JS for frontend.
* GeoPy, Folium, GeoIP and GeoDjango for "trip planning" module
* HTMLCalendar for managing the activities [Source](https://www.huiwenteo.com/normal/2018/07/24/django-calendar.html)
